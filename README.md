# ParduGO web

## setup:
- install dependencies\
    `yarn` / `yarn install`
- copy .env from example `cp .env.local.example .env.local` and change values 
- run dev server  `yarn dev` (http://localhost:3000)

## build
```bash
yarn build
yarn start
```

## technologies used:
- Next.js
- Typescript
- Tailwindcss
- GraphQL API
